    mixins

    font-size

    @mixin font-size($font-size, $line-height: auto, $modifier: 0) { ... }

    Description

    Generates a rem font-size and a baseline-compatible unitless line-height from font-size value. You can also modify the line-height by increments, while staying in the baseline grid, by setting the $modifier parameter.

    It takes a positive or negative number, and it will add or remove "lines" to the generated line-height. This is the recomended way to do it, unless you really need an absolute value.

    It is recommended to only add or subtract full integers or half steps.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $font-size— noneNumber— none
    $line-height

    Line height

    		Numberauto
    $modifier

    Line modifier

    		Number0

    Example

    Example 1:

    @include font-size(1.25rem);

    Example 2, force line-height:

    @include font-size(1rem, 1.5);

    Example 3, add 2 lines:

    @include font-size(2rem, $modifier: +2);

    Example 4, subtract -0.5 lines:

    @include font-size(0.875rem, $modifier: -0.5);

    Throws

    • #{$font-size} needs to be a number.

    • Dâ€™oh! #{$line-height} is not a valid value for $line-height.

    Requires

    • [variable] global-font-size
